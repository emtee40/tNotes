## FolderModel API

- [addNote](#addNote)
- [removeNote](#removeNote)
- [clearNote](#clearNote)
- [updateModifiedTime](#updateModifiedTime)

---------------------------------
### addNote

概要：`addNote(id)`

描述：添加笔记

参数：

- id(number)：添加笔记的id

### removeNote

概要：`removeNote(id)`

描述：删除笔记

- id(number)：删除笔记的id

### clearNote

概要：`clearNote(id)`

描述：清空所有笔记

### updateModifiedTime

概要：`updateModifiedTime(time)`

描述：更新“修改时间”

- time(number)：修改时间
