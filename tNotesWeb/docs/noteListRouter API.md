## noteListRouter API

- [newNote](#newNote)
- [showNote](#showNote)
- [setSelectedFolder](#setSelectedFolder)

-------------------

### newNote

概要：`newNote()`

描述：新建笔记

### showNote

概要：`showNote(id)`

描述：选择笔记

参数：

- id(number)：选择的笔记的id

### setSelectedFolder

概要：`setSelectedFolder(NotesList)`

描述：接收folder传来的Note数组，并显示在Note列表中

- NotesList(Note)：folder传过来的Note类型数组
